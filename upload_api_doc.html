7. Get Main Playback File

Retrieve the master playlist.

Endpoint: /api/play/{uploadId}?expires={expires}&signature={signature}
 Method: GET

Headers

  • accept: */*
  • api-key: OMpqVWAH.UC80wyXTtPwhDgAUdCTx6

Route Parameters

  • uploadId: string ✔ → Upload task ID
  • expires: long ✔ → Expiration timestamp
  • signature: string ✔ → Generated signature

Signature Generation Method (C#)

var requestPath = $"v/{uploadId}/v1/master.m3u8";

var url= $"{requestPath}?expires={expires}";

using (var hmac = new HMACSHA256(Encoding.UTF8.GetBytes(secretKey)))

{

    var hash = hmac.ComputeHash(Encoding.UTF8.GetBytes(url));

    return Convert.ToBase64String(hash).Replace("+", "-").Replace("/", "_").Replace("=", ""); // URL-safe

}
